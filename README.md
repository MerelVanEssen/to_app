# To_App

**To_App** is een eenvoudige **to-do applicatie** ontwikkeld voor **VRcafe Haarlem**. Hiermee kunnen gebruikers hun taken overzichtelijk beheren en bijhouden.

---

## Features
- Taken toevoegen, verwijderen en afvinken  
- Lokale opslag van taken  
- Overzichtelijke en eenvoudige interface  

---

## Installatie & Gebruik

1. Installeer de benodigde dependencies:  
   ```bash
   npm install

2. Start server:
   npm run dev

3. Open in browser:
   http://localhost:4321
